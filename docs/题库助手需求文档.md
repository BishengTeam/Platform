# 题库助手需求文档

> 版本：V1.0
> 日期：2026-06-08
> 范围：小程序前端（Platform）+ 后端（Backend）

---

## 一、概述

题库助手是小程序核心功能模块，提供刷题练习、模拟考试、错题本、收藏、打卡签到、练习统计等能力。题型仅限选择题（单选 + 多选 + 判断）。

前端路由：`pages/quiz/*`

## 三、后端待开发接口

### 3.1 分类题目计数（P0 — 必须）

**现状**：`GET /api/quiz/categories` 返回的分类数据不包含题目数量，前端只能 mock。

**需求**：分类响应增加 `question_count` 字段，统计该分类下的题目总数（含子分类）。

```json
// 响应示例
{
  "id": 1,
  "name": "H3CNE",
  "parent_id": null,
  "description": "H3CNE 认证题库",
  "question_count": 500,
  "children": [
    {
      "id": 2,
      "name": "网络基础",
      "parent_id": 1,
      "question_count": 150,
      "children": []
    }
  ]
}
```

**实现建议**：在 `QuizService.list_categories()` 中 JOIN `quiz_question` 表，用 `func.count()` 子查询或 Python 聚合统计每个分类（含子分类递归汇总）的题目数。

### 3.2 练习统计（P0 — 必须）

**需求**：新增 `GET /api/quiz/stats`，返回当前用户的练习统计数据。

**认证**：需登录

**响应字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `total_answers` | int | 累计答题数（按 QuizRecord 去重计数） |
| `correct_answers` | int | 答对数 |
| `accuracy` | float | 正确率（百分比，如 67.5） |
| `total_questions` | int | 题库总题数 |
| `answered_questions` | int | 已答题数（唯一题数） |
| `completion_rate` | float | 完成率（百分比） |
| `streak_days` | int | 当前连续打卡天数 |
| `total_checkin_days` | int | 累计打卡天数 |
| `wrong_count` | int | 当前错题本题目数 |
| `collected_count` | int | 当前收藏题目数 |
| `today_answers` | int | 今日答题数 |
| `today_correct` | int | 今日答对数 |

```json
// 响应示例
{
  "code": 0,
  "data": {
    "total_answers": 328,
    "correct_answers": 220,
    "accuracy": 67.1,
    "total_questions": 1000,
    "answered_questions": 280,
    "completion_rate": 28.0,
    "streak_days": 7,
    "total_checkin_days": 45,
    "wrong_count": 32,
    "collected_count": 18,
    "today_answers": 15,
    "today_correct": 11
  }
}
```

**SQL 参考**：

```sql
-- 总答题数
SELECT COUNT(*) FROM quiz_record WHERE user_id = :uid;

-- 答对数
SELECT COUNT(*) FROM quiz_record WHERE user_id = :uid AND is_correct = true;

-- 已答题数（唯一）
SELECT COUNT(DISTINCT question_id) FROM quiz_record WHERE user_id = :uid;

-- 题库总题数
SELECT COUNT(*) FROM quiz_question;

-- 今日答题
SELECT COUNT(*) FROM quiz_record 
WHERE user_id = :uid AND DATE(updated_at) = CURRENT_DATE;
```

### 3.4 模拟考试（P0 — 必须）

**需求**：新增模拟考试会话管理，支持限时作答和结果统计。

#### 3.4.1 创建考试 `POST /api/quiz/exam/start`

**认证**：需登录

**请求体**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `category_id` | int | 否 | 限定分类，不传则全题库随机 |
| `question_count` | int | 是 | 抽题数量（建议 50-100） |
| `duration_minutes` | int | 否 | 考试时长（分钟），默认 60 |
| `question_type` | str | 否 | 限定题型：single_choice / multiple_choice / judge |

**响应**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `exam_id` | int | 考试会话 ID |
| `questions` | list | 题目列表（不含正确答案） |
| `total` | int | 总题数 |
| `duration_seconds` | int | 考试时长（秒） |
| `started_at` | str | 考试开始时间 |

#### 3.4.2 提交考试 `POST /api/quiz/exam/submit`

**请求体**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `exam_id` | int | 考试会话 ID |
| `answers` | list | 答案列表，每项 `{ question_id, user_answer }` |
| `elapsed_seconds` | int | 实际用时（秒） |

**响应**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `exam_id` | int | 考试会话 ID |
| `total` | int | 总题数 |
| `correct_count` | int | 答对数 |
| `wrong_count` | int | 答错数 |
| `unanswered_count` | int | 未答数 |
| `score` | float | 得分（百分制） |
| `accuracy` | float | 正确率 |
| `elapsed_seconds` | int | 实际用时 |
| `details` | list | 逐题结果（含正确答案和解析） |

#### 3.4.3 考试记录 `GET /api/quiz/exam/history`

**认证**：需登录

**查询参数**：`page`、`page_size`

**响应**：分页考试记录列表，每项含考试时间、题数、得分、正确率。

#### 3.4.4 考试详情 `GET /api/quiz/exam/{exam_id}`

**认证**：需登录

**响应**：某次考试的完整结果（含逐题对错和解析）。

#### 数据模型建议

新增 `quiz_exam` 表：

```sql
CREATE TABLE quiz_exam (
    id SERIAL PRIMARY KEY,
    user_id INT NOT NULL REFERENCES "user"(id),
    question_ids JSONB NOT NULL,          -- 考试题目 ID 列表
    answers JSONB,                         -- 用户答案 {"q1": "A", ...}
    total INT NOT NULL,
    correct_count INT,
    wrong_count INT,
    score DECIMAL(5,2),
    elapsed_seconds INT,
    duration_seconds INT NOT NULL,
    started_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    submitted_at TIMESTAMPTZ,
    status VARCHAR(16) NOT NULL DEFAULT 'in_progress'  -- in_progress / completed / timeout
);
```
