# 甲方演示方案

## 一、演示目标

向甲方展示「智天远微信小程序平台」的核心功能：**AI 智能助手对话** 与 **五大专区（考试/学习/竞赛/活动/就业）**，体现平台的完整性和可用性。

## 二、演示流程（约 8-10 分钟）

### 阶段 1：启动与首页（30 秒）

| 步骤 | 操作 | 展示内容 |
|------|------|---------|
| 1.1 | 打开小程序 | 授权登录页（一键微信登录） |
| 1.2 | 点击登录 | 进入首页「智能助手」，展示 WelcomeCard（5 个专区快捷入口）+ AI 欢迎语 |

### 阶段 2：AI 对话演示（3-4 分钟）—— 核心亮点

按以下脚本依次演示（可硬编码模拟对话）：

```
用户：(点击快捷问题) "如何报名参加认证？"

AI：  您好！H3CNE 认证考试报名流程如下：
      📋 第一步：确认报考条件（无学历/经验限制）
      📋 第二步：选择考试中心（全国 200+ 考点）
      📋 第三步：在线缴纳考试费 ¥1,200
      📋 第四步：预约考试时间
      
      [考试专区卡片] → 可点击跳转
      
      需要我帮您推荐备考课程吗？

---

用户：(点击快捷问题) "有什么学习课程推荐？"

AI：  根据 H3CNE 考试大纲，为您推荐以下课程：
      📚 《H3CNE 网络基础》- 入门必学
      📚 《路由交换技术详解》- 核心考点
      📚 《H3CNE 认证冲刺班》- 考前突击
      
      [学习专区卡片] → 可点击跳转

---

用户：(输入) "最近有什么竞赛？"

AI：  近期有 3 个进行中的竞赛：
      🏆 全国大学生网络技术大赛（进行中）
      🏆 H3C 杯数字技术精英赛（进行中）
      🏆 新华三网络技能挑战赛（即将开始）
      
      [竞赛专区卡片] → 可点击跳转

---

用户：(输入) "帮我找一下老师"

AI：  为您匹配到 H3CNE 认证讲师：
      👨‍🏫 张老师 - H3CNE 金牌讲师 / 考官
      📱 微信：H3C_TeacherZhang
      📞 电话：138-1234-5678
      
      [复制微信号] 按钮
```

**关键演示点：**
- 快捷问题入口
- AI 多轮对话能力
- 富文本卡片回复（考试卡片、课程推荐、讲师名片）
- 专区快捷跳转

### 阶段 3：专区浏览（2-3 分钟）

| 步骤 | 操作 | 展示内容 |
|------|------|---------|
| 3.1 | 点击底部「专区」Tab | 专区首页：顶部 Banner 轮播 + 5 个专区入口网格 |
| 3.2 | 点击「考试专区」 | Banner 轮播 → 标签筛选（全部/新华三/华为/深信服/软考）→ 考试卡片列表 |
| 3.3 | 切换标签「华为」 | 列表过滤展示华为相关考试 |
| 3.4 | 返回，进入「学习专区」 | Banner → 标签筛选（基础/进阶/实战/认证）→ 课程卡片列表 |
| 3.5 | 快速展示竞赛专区 | 进行中/即将开始/已结束 三类竞赛 |

### 阶段 4：个人中心（1-2 分钟）

| 步骤 | 操作 | 展示内容 |
|------|------|---------|
| 4.1 | 点击「我的」Tab | 头像、昵称、备考状态、订单状态栏 |
| 4.2 | 点击「我的订单」 | 全部/待付款/已报名/已取消 四个标签筛选 |
| 4.3 | 展示「证书中心」 | H3CNE/H3CSE 证书卡片 |
| 4.4 | 展示「消息通知」 | 已读/未读通知列表 |

## 三、代码调整清单

### 3.1 AI 对话增强（核心）

**文件：`src/types/message.ts`** — 扩展卡片类型
- 新增 `ExamCard`、`CourseCard`、`ZoneLinkCard` 类型
- `Message.card` 支持联合类型

**文件：`src/constants/strings.ts`** — 丰富 AI 回复内容
- 替换 `INDEX_AI_DEFAULT` 为真实的考试/课程/竞赛回复文案
- 新增多条模拟回复模板

**文件：`src/pages/index/index.tsx`** — 多轮对话逻辑
- 增加关键词匹配（认证、报名、考试、课程、学习、竞赛、活动、就业、老师）
- 根据关键词返回不同的富文本+卡片回复
- 支持上下文关联（如先问考试再问费用）

**文件：`src/components/ChatArea/index.tsx`** — 渲染富文本卡片
- 新增考试卡片、课程推荐卡片、专区链接卡片的渲染

**文件：`src/constants/mock/home.ts`** — 丰富快捷问题
- 替换为更自然的演示用问题

### 3.2 演示体验优化

### 3.3 专区数据完善

- 考试专区：6 条数据 + 5 个标签（已完备）

## 六、风险与注意事项

1. **演示环境**：建议使用微信开发者工具的真机调试模式，确保小程序运行流畅
2. **网络依赖**：当前全部使用本地 mock 数据，无需网络，演示不受网络环境影响
3. **版本锁定**：演示前确认代码已提交，避免演示中出现意外改动
4. **备用方案**：如遇工具崩溃，可切换至浏览器 H5 模式预览
